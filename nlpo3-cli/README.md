# nlpo3-cli

Command line interface for nlpO3, a Thai natural language processing library.

## Install

```bash
cargo install nlpo3-cli
```

## Usage

```bash
nlpo3 help
```

## Example

```bash
echo "ฉันกินข้าว" | nlpo3 segment
```
